# decision support system for selecting the best teachers in schools
A system that assists in decision-making for selecting the best teacher in school.

<p>This system provides 3 user roles.</p>
<ul>
  <li>Admin</li>
  <li>evaluator</li>
  <li>Guest</li>
</ul>
